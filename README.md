# Лабораторная работа 2 по структурам данных

Для работы программы используется библиотека OpenBLAS.
Скачать OpenBLAS: https://github.com/OpenMathLib/OpenBLAS/releases

### Подключить include:
Проект → Свойства → C/C++ → Общие → Дополнительные каталоги включаемых файлов

### Подключить lib:
Проект → Свойства → Компоновщик → Общие → Дополнительные каталоги библиотек

### Добавить библиотеку:
Компоновщик → Ввод → Дополнительные зависимости

### Включить OpenMP:
/C++ → Язык → Поддержка OpenMP → Да (/openmp)
